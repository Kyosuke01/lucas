# 🔒 Politique de sécurité — LUCAS

## Versions supportées

| Version | Supportée |
|---|---|
| main | ✅ |

## Signaler une vulnérabilité

Ne **pas** créer d'issue publique pour les failles de sécurité.

Merci de signaler les vulnérabilités via **GitHub Security Advisories** :
[Créer un rapport privé](https://github.com/Kyosuke01/lucas/security/advisories/new)

Ou par email à l'adresse indiquée dans le profil GitHub du mainteneur.

### Ce qu'il faut inclure

- Description de la vulnérabilité
- Étapes pour reproduire
- Impact potentiel
- Version affectée
- Correctif suggéré (facultatif)

Nous nous engageons à répondre sous **72 heures** et à publier un correctif dans les meilleurs délais.
